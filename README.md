# Shehryar Muhammad Aslam — Portfolio

A clean, responsive personal portfolio website built with **HTML, CSS, and JavaScript**.

## Features

- Responsive layout for desktop and mobile
- Dark/light theme with local storage
- Animated hero text
- Scroll reveal animations
- Particle background
- Contact form using `mailto:`
- Ready for GitHub Pages deployment

## Pages

- `index.html` — Home
- `about.html` — About section
- `skills.html` — Skills and tools
- `projects.html` — Project showcase
- `contact.html` — Contact information

## How to Run

Open `index.html` directly in a browser, or use a local server:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## GitHub Pages Deployment

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Go to **Settings > Pages**.
4. Select **Deploy from branch**.
5. Choose `main` branch and `/root` folder.
6. Save and wait for the live link.
